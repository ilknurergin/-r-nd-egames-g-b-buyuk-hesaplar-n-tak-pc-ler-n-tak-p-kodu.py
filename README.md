🚀 Bluesky Targeted Growth Bot (Interest-Based Follower)
Bu proje, Bluesky üzerindeki Firehose (canlı veri akışı) mekanizmasını kullanarak, belirlediğiniz ilgi alanlarına (örneğin: indie games) sahip kullanıcıları anlık olarak tespit eder ve etkileşime girer. Paylaşılan ilgi alanları sayesinde geri takip (follow-back) oranını maksimize eden akıllı bir büyüme aracıdır.

📌 Özellikler
Canlı Takip: Belirlediğiniz anahtar kelimeleri Firehose üzerinden saniyelik olarak tarar.

Hedef Odaklı: Sadece sizinle aynı ilgi alanına sahip gerçek kullanıcıları bulur.

Zahmetsiz Büyüme: "Takibe takip" mantığını manuel işlem yapmadan otomatiğe bağlar.

Hız ve Performans: atproto kütüphanesi kullanılarak yüksek hızda işlem yapacak şekilde optimize edilmiştir.

🛠️ Kurulum
Depoyu Klonlayın:

Bash
git clone https://github.com/kullaniciadi/reponame.git
cd reponame
Gerekli Kütüphaneleri Yükleyin:

Bash
pip install atproto
Yapılandırma:
Kod içerisindeki aşağıdaki alanları kendi bilgilerinizle doldurun:

HANDLE: Bluesky kullanıcı adınız.

APP_PASSWORD: Bluesky uygulama şifreniz.

KEYWORDS: Takip etmek istediğiniz ilgi alanları (Örn: "indie games", "python", "art").

🚀 Kullanım
Botu başlatmak için terminale şu komutu yazın:

Bash
python main.py
⚠️ Önemli Uyarı
Bu araç eğitim ve kişisel kullanım amaçlıdır. Bluesky'ın topluluk kurallarına ve spam politikalarına uymak kullanıcının sorumluluğundadır. Aşırı kullanım hesap kısıtlamalarına yol açabilir; bu yüzden bekleme sürelerini (time.sleep) dikkatli ayarlamanız önerilir.
